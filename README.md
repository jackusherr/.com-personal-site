2026-01-22: jackusher.com. HTML-only personal site, now with a simple Hugo blog section.

## Add a blog post

1. Create a new file:
   `hugo new posts/my-new-post.md`
2. Edit the generated file in `content/posts/`.
3. Build/deploy.

Posts are listed at `/posts/` and each post is published at `/posts/<slug>/`.
